HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The !DOCTYPE html element tells the browser of what type of document it should expect when opened
  • The html element is the start or root of our html document
  • The lang="en" element tells the browser the language of the document
  • The title element defines the title of the webpage
  • The style element defines the CSS elements related to a website
  • The link element connects an external resource to the html document
  • The metadata element defines metadata like character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • The charset element or or character set, as UTF-8, which allows us to use most characters and symbols on the page
  • The http-equiv element sets an HTTP header for content.
  • The viewport element sets the default view for our webpage.
  • The div element creates division on a webpage
  • The span element adds an inline division on a webpage
  • The h1 h2 h3 h4 h5 and h6 elements are the level headings of a given text block
  • The p element represents a block of text
  • The ul means unordered ol means ordered and li means list items. All represent numbered or bulleted lists
  • The img elements contains the information on the webpage's image
  • The src element idicates the location of the image
  • The br element is a line between two blocks of content
  • The a element creates links on the same webpage or other webpages
  • The href element points to the URL of the link
  • The id identifies the elements
  • The class element identifies specific elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Width is the width of the webpage
  • Height is the height of the webpage
  • border is the border of the webpage
  • Selectors are patterns used to select elements you want to style
  • Declarations is the property and value of the CSS property
  • Color is the property that applies color to the elements
  • Text-align is a positioning property
  • Display is a property which determines the behavior of the webpage's elements

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Main branch: the default branch where the final updated version of the application is
  • Feature branch: A copy of the main branch where commits and changes are made
  • Pull request: Merging a comparison branch into the base branch
  • git add -A: Idicates add all changes
  • git commit -m: a description of a recent change in the project
  • git pull: receives changes into the local branch
  • git push: pushes changes from the local branch to the Github branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • var is used to declare a variable
  • let declares a variable when the variable changes
  • const declares a variable when it is a general variable
  • = assigns the data to the variable
  • script.js is a JavaScript file
  • A string contains words in quotation marks
  • console.log() is method used to output a message to the web console by adding an argument
  • An if statement introduces the code. The code will only execute if it meets a certain condition or true
  • An else statement is a block of code to be executed if the same condition is false
  • An else if is a block of code to be executed if the first condition is false
  • === checks to see if two values equal each other and if so states True or False if not equal
  • An array holds a group of related data in [] brackets
  • An index is the location numbers of the individual datas in an array
  • A for loop uses a pattern to perform the task on all items in an array and executes the code over and over
  • A function is a set of instructions that tells a computer on how to do a certain task
  • An undefined occurs when a return statement returns no value
  • A switch statement is used to perform different actions for certain conditions
